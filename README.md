# Installation
```
    sudo wget https://raw.githubusercontent.com/indivisualvj/symfony-init/master/symfony-init -O /usr/local/bin/symfony-init && sudo chmod +x /usr/local/bin/symfony-init 
```

```
    sudo curl https://raw.githubusercontent.com/indivisualvj/symfony-init/master/symfony-init >> /usr/local/bin/symfony-init && sudo chmod +x /usr/local/bin/symfony-init 
```
